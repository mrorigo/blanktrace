# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-23

### Added
- Initial release of BlankTrace.
- MITM HTTP/HTTPS Proxy functionality.
- Fingerprint randomization (User-Agent, Accept-Language).
- Cookie blocking and stripping.
- Flexible cookie configuration (allow/block lists).
- Periodic database cleanup (configurable retention).
- Tracker blocking with regex support.
- SQLite logging for requests and tracking data.
- CLI for management and statistics.
