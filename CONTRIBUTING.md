# Contributing to BlankTrace

Thank you for your interest in contributing to BlankTrace! We welcome contributions from the community to help make this project better.

## How to Contribute

1.  **Fork the repository**: Create a fork of the repository on GitHub.
2.  **Create a branch**: Create a new branch for your feature or bug fix.
    ```bash
    git checkout -b feature/my-new-feature
    ```
3.  **Make your changes**: Implement your changes, ensuring you follow the coding style and conventions.
4.  **Run tests**: Make sure all tests pass before submitting your changes.
    ```bash
    cargo test
    ```
5.  **Submit a Pull Request**: Open a Pull Request against the `main` branch. Provide a clear description of your changes and why they are needed.

## Coding Standards

-   Follow standard Rust coding conventions (use `cargo fmt`).
-   Write clear and concise commit messages.
-   Add documentation for new features or changes.
-   Ensure no sensitive information is committed.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible, including steps to reproduce the issue.
